## NotebookLM Analysis Workflow

### Phase 1: Initial Video Analysis
- Select 5-10 representative videos from this list
- Upload each URL to NotebookLM project
- Use 'Generate' to create summaries
- Use 'Ask' to query specific trading strategies

### Phase 2: Strategy Pattern Mining
- Create comparative analysis across videos
- Use 'Ask' with questions like:
  - 'What technical analysis patterns does SMB Capital use?'
  - 'What risk management techniques are recommended?'
  - 'What are the most common entry/exit signals?'

### Phase 3: Market Context
- Analyze videos by publication date to understand market context
- Track how strategies evolved over time
- Identify successful/unsuccessful trade examples

## Export Format

After analysis, export NotebookLM responses as:
- Markdown (.md) for integration with this KB
- JSON for programmatic processing
- PDF for documentation
